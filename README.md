# Lender Matching Platform

## Overview

The **Lender Matching Platform** is a system that matches borrower loan applications with lenders based on predefined underwriting policies.

Borrowers submit loan application details such as credit score, revenue, loan amount, and industry. The platform evaluates these inputs against lender policies and returns a list of lenders that satisfy the criteria.

This project demonstrates a simplified version of a loan underwriting and matching engine similar to those used by fintech marketplaces.

---

## Technologies Used

* **Backend:** FastAPI (Python)
* **Frontend:** TypeScript React, Tailwind CSS
* **Database:** PostgreSQL
* **ORM / Database Toolkit:** SQLAlchemy + Pydantic
* **Workflow Orchestration**: Python + Hatchet
* **Package Management:** pip / npm
* **Version Control:** Git + GitHub

---

## Features

* Loan application submission and detailed view
* Parsing and normalizing lender policies into structured format
* Lender policy management (view, add, edit policies)
* Matching Engine to evaluate loan applications against each lender's criteria
* Well-defined workflow to add more lenders (through PDF guidelines)
